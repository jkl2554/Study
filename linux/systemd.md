```
# systemctl --user 유저별 서비스 설정

# /etc/systemd/system/ # root systemd 디렉토리
# /usr/lib/systemd/user/ # user systemd 디렉토리
# ~/.config/systemd/user # user custom systemd 디렉토리
```
podman 은 서비스 generate 가능

```
# podman generate systemd keycloak  > /etc/systemd/system/podman_keycloak.service -> Podman 서비스 생성(root)  systemctl enable podman_keycloak
# podman generate systemd keycloak  > ~/.config/systemd/user/podman_keycloak.service -> Podman 서비스 생성(user) systemctl --user enable podman_keycloak
```
